Syntax: variable_name | filter_name

capitalize the first character of the value

Name: {{name | capfirst }}

convert a string into titlecase by making words start with an uppercase character

Name: {{name | title }}

return a slice of the list

Name: {{name | slice:"1:5" }}

remove all values of arg from the given string

Name: {{name | cut:" " }}

add an argument to the value

Roll: {{roll | add:"240"}}

Today: {{today}}

format a date according to the given formats

Today: {{ today | date:"D d M Y" }}

Time: {{ today | time:"H:i" }}

return the length of the value

Courses Length: {{ courses | length }}

takes a list of dictionaries and returns that list sorted by the key given in the argument

Courses sorted by title: {{ courses | dictsort:"title" }}

join a list with a string

Join List: {{ lst | join:'+' }}
