var: variables declared with var are function scoped. their scope is only inside the function in which they are declared. And cannot be accessed outside of the function.

let: variables declared with let are block scoped. their scope is only inside the block and is mutable in which they are declared. And cannot be accessed outside of the block.

const: variables declared with const are block scoped. their scope is only inside the block in which they are declared. And cannot be accessed outside of the block. But variables declared with const are immutable.

null: null is used as a value of a variable which is not initialized yet or in case of absance of value.

undefined: when a variable is declared but not assigned any value, then the default value is undefined. And when we try to access an object property that does not exist or to access an array element beyond the length of the array, the result is undefined.

REST API: Representational State Transfer Application Programming Interface, is a set of rules and conventions for building and interacting with web services. Which is a way for two computer systems to communicate over the internet.
